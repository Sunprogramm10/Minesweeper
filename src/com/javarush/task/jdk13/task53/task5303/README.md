# Змейка (Snake Game) — игровой движок JavaRush

Классическая игра **Змейка**, реализованная на Java в рамках учебной задачи JavaRush.


## 🎯 Цель игры
Управляй змейкой, собирай еду (яблоки), избегай столкновений со стенами и собственным хвостом!


## 🚀 Как запустить
### Вариант 1 — в JavaRush (самый простой)
1. Залогинься на [javarush.com](https://javarush.com)
2. Открой проект / задачу по ссылке: https://javarush.com/projects/apps/504649
3. Запусти → играй прямо в браузере

### Вариант 2 — локально в IntelliJ IDEA
1. Установи плагин **JavaRush** (Settings → Plugins → Marketplace → "JavaRush" → Install → Restart)
2. Скопируй классы `Snake.java` , `SnakeGame.java`, `Apple.java` и `Direction.java` в проект
3. Правой кнопкой на `SnakeGame` → **Run 'SnakeGame.main()'**


## ✨ Особенности реализации
- Змейка состоит из головы и сегментов тела
- Движение по стрелкам (← ↑ → ↓)
- Запрет разворота на 180° (нельзя резко развернуться назад)
- Голова зелёная, если жива; красная — при проигрыше
- Тело меняет цвет при смерти
- Поле 15×15 клеток
- Автоматическое перемещение по таймеру
- Рисование с помощью `setCellValueEx` (эмодзи голова 🐲 / тело ⚫)

## 🕹️ Управление
| Клавиша       | Действие                  |
|---------------|---------------------------|
| ← (LEFT)      | Движение влево            |
| → (RIGHT)     | Движение вправо           |
| ↑ (UP)        | Движение вверх            |
| ↓ (DOWN)      | Движение вниз             |
